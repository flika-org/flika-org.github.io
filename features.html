Features

Tools and capabilities of FLIKA image processing

Easy install

Once Python (2.* or 3.*) is installed, download and extract the zip file, run Flika.bat and all plugins will be automatically installed before startup.

Open Source

FLIKA and its Python source code are freely available and in the public domain. No license is required yet.

Scripts/Macros

Write your own scripts or record them from preprocessed images to automate tasks. Create custom tools using the script editor or find existing tools in the script database.

Plugins

Extend FLIKA by developing plugins. Write your own plugins in Python and include them in the plugins folder for easy testing. Send us your a link to your plugin to add it to the FLIKA plugin library for all users to install.

Quick Image Handling

Read, preview, and manipulate files in a matter of seconds. Python libraries backended enable FLIKA to handle large files surprisingly fast.

Support for Filetypes

Open and save all supported data types as TIFF (uncompressed) or as raw data. Save movies as MP4s. Open TIFFs, ND2s, STKs, and more. Use plugins to import/export files of any kind.

Interactive Image Display

Written with pyqtgraph, FLIKA enables easy pan and zoom capabilities alongside a histogram of pixel values for easy range selection.

Regions of Interest

Create rectangular and irregular area selections, or line and point selections. ROIs can be drawn, moved, measured, copied to other images, and imported/exported for future use.

Image Processing

Apply filters (butterworth, gaussian, Fourier, etc.), add/subtract with other images, divide by a subset of frames, average a stack of images by pixel or by frame, and several other analysis tools.
