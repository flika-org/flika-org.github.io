Flika

An interactive image processing tool for biologists, written in Python for simple, tunable code in an intuitive yet powerful program.

Install

Filetypes

Supports numerous types of image filetypes: .tif, .nd2, .stk, and more.

Script Playback

Record the steps of your analysis in a script to automate a series of Flika commands.

Plugin Support

Write your own plugins for Flika to maximize productivity.

Always Improving

Flika was created to automatically identify and automate the analysis of local Ca2+ signals but has grown into a general analysis tool for studying biological images
Ellefsen, K., Settle, B., Parker, I. & Smith, I. An algorithm for automated detection, localization and measurement of local calcium signals from camera-based imaging. Cell Calcium. 56:147-156, 2014
