Always Improving

Flika was created to detect and measure Ca2+ puffs caused by IP3 channel openings but has grown into a general analysis tool for biological movies.

Perfect For Biologists

Written and maintained by the Parker Lab, FLIKA is written to simplify lab analysis in a user friendly way with no prior programming knowledge necessary. Intuitive layout modeled after ImageJ layout.

Github Maintained

All code is currently available on Github, along with past versions. Keep up to date and install through your github account to get FLIKA updates as they are released.
