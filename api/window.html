Module: window¶

class flika.window.Window(tif, name='Flika', filename='', commands=[], metadata={})¶

Window objects are the central objects in flika. Almost all functions in the process module are performed on Window objects and output Window objects.

imageArray()¶

returns image as a 3d array, correcting for color or 2d image

paste()¶

This function pastes an ROI from one window into another. The ROIs will be linked so that when you translate one of them, the other one also moves

About flika

flika is an image processing program for biologists written in Python.
